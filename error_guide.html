교육용 최소설치 또는 특정 Windows 라이선스 에디션에서 발생. 기능 자체가 이미 내장되어 있으므로 무시해도 됩니다.
87 ERROR_INVALID_PARAMETER (DISM) 무시가능
해당 Windows 에디션에 지정한 기능이 없거나 이미 내장되어 별도 활성화가 불필요함.
교육용 최소설치 Windows에서는 COM+ 기능이 빌트인된 경우 이 코드가 나옵니다.
4 DllRegisterServer 진입점 없음 무시가능
해당 DLL이 DllRegisterServer 함수를 export하지 않아 regsvr32로 등록 불가.
COM 자가등록 방식이 아닌 DLL입니다. 건너뜁니다.

주의가 필요한 코드

1 COM+ 등록 실패 (RegSvcs) 오류
System.EnterpriseServices.dll COM+ 카탈로그 등록 실패.

원인 1: COMSysApp 서비스 중지 상태 → STEP5 재시작 후 PC 재시작 시 자동 재등록.
원인 2: COM+ 카탈로그 손상 → 아래 수동 복구 참고.
원인 3: Windows Sandbox 환경 제한 → 실제 PC에서는 정상 동작.
2 ERROR_FILE_NOT_FOUND (서비스 시작) 오류
서비스가 의존하는 DLL 또는 바이너리를 찾을 수 없음.

⚠ COMSysApp start: 2 는 SmartMaker 오류의 직접 원인입니다.
COMSysApp 없이는 RegSvcs가 COM+ 카탈로그에 접근 불가 → RegSvcs 전부 실패.

COMSysApp 복구 순서:
1. 관리자 CMD: sfc /scannow (시스템 파일 복구, 약 10분)
2. 재시작 후 배치 재실행
3. 여전히 실패 시: dism /online /cleanup-image /restorehealth
4. 로그의 [DIAG] 섹션에서 sc qc COMSysApp 결과로 서비스 경로 확인

MSDTC start: 2: 관리자 CMD에서 msdtc -install 수동 실행.
5 ERROR_ACCESS_DENIED 오류
관리자 권한 부족 또는 다른 프로세스가 해당 파일 사용 중.
배치 파일을 우클릭 → 관리자 권한으로 실행하세요.

RegSvcs 계속 실패 시 — 수동 복구 순서
